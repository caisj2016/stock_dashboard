# 日本股票观察系统用户指南（假数据示例版）

说明：

- 本文档用于演示当前系统已经完成的功能。
- 文中的股票、价格、盈亏、信号等内容为假数据示例，仅用于说明界面和操作流程。

## 1. 系统概览

当前系统包含 3 个主要页面：

1. 主页面
   用于查看持仓卡片、市场摘要、个股资讯、右侧自选列表。

2. 技术选股页面
   用于按技术策略筛选股票，并查看候选股票图表。

3. 个股图表页面
   用于查看单只股票 K 线、成交量、MACD、个股资讯，以及右侧自选切换。

## 3. 主页面功能

主页面已经完成的功能：

- 持仓卡片显示
  - 股票代码
  - 日文名称
  - 最新价格
  - 涨跌幅
  - 持股数 / 成本价 / 市值 / 盈亏

- 新闻模块
  - 个股最新资讯
  - 日经简报
  - 半导体简报

- 右侧自选列表
  - 支持 `已持有 / 观察中 / 全部`
  - 支持添加股票
  - 支持单击、双击、右键菜单

## 4. 技术选股页面功能

技术选股页已经支持：

- 股票池切换
  - 核心 45
  - Nikkei 225
  - TOPIX Core 30

- 策略切换
  - 超卖反弹
  - MACD 金叉
  - 放量上涨
  - 均线突破
  - 综合信号

- 结果卡片查看
  - 候选股票
  - 价格与涨跌
  - 命中信号
  - 图表预览

- 右侧共享自选列表
  - 与主页面、图表页保持统一

## 5. 个股图表页面功能

个股图表页已经支持：

- K 线主图
- 成交量开关
- MACD 开关
- 周期切换
  - 日线
  - 周线
  - 月线
  - 4 小时

- 左侧工具栏
  - 普通查看
  - 趋势线
  - 水平线
  - 斐波那契回撤
  - 价格范围
  - 日期范围
  - 清空画线

- 个股资讯区
  - 显示当前股票相关新闻

- 右侧共享自选列表
  - 单击股票：左侧 K 线图同步切换
  - 双击股票：进入该股票图表页
  - 右键股票：打开菜单

## 6. 右侧自选模块交互

当前统一交互规则如下：

### 单击

- 在右侧自选列表中单击一只股票
- 右下方详情卡显示该股票摘要信息
- 如果当前在图表页，左侧 K 线也会同步切换

### 双击

- 双击自选列表中的股票
- 直接进入该股票图表页

### 右键

右键菜单当前有 2 个功能：

1. 编辑状态
   可修改：
   - 已持有 / 观察中
   - 持股数
   - 成本价

2. 加入观察
   将当前股票快速切换为观察中

## 7. 数据保护

系统已经加入基础数据保护：

- 持仓数据保存在 `portfolio.json`
- 保存时采用原子写入，减少损坏风险
- 每次保存前自动备份
- 备份保存在 `data_backups/`
- 主文件损坏时会尝试使用最近一份备份恢复

## 8. 建议使用流程

推荐的使用顺序：

1. 在主页面查看持仓与新闻摘要
2. 在技术选股页面寻找新机会
3. 打开个股图表页进行细看
4. 通过右侧自选列表快速切换股票
5. 用右键菜单维护状态和观察列表
